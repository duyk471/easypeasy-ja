# 第14章　Casual User カジュアルユーザー、軽度のポルノ使用者

重度のポルノ使用者は、軽度のポルノ使用者を妬む傾向にある、皆、こういう事を言っている人を見たことがある：「え、自分は一週間自慰をしなくても平気だよ」我々は、自分もそうだったらなあと願う。信じられないかもしれないが、ポルノ使用者は、誰もポルノ使用者であることを良く思っていない。忘れないで：

　ポルノ使用者は誰も、軽度であれ重度であれ、ポルノ使用者になるという決断をしない、＜ゆえに＞、

　　すべてのポルノ使用者は自分が間抜けだと感じている、＜ゆえに＞、

　　　すべてのポルノ使用者は、自分自信と他者に嘘を付く必要がある、自分の間抜けさを正当化するという不毛な試みのために。

ゴルフ狂の人は自分がどれだけプレイしたか、どれだけプレイしたいかを誇示する、ならばなぜポルノ使用者は自分の自慰回数の少なさを誇示しないのか？もしこれが真の基準であるならば、称賛すべきは、間違いなく、自慰を全くしないことのはずだ、そうでしょう？

誰かがあなたに、「自分は一週間人参を食べなくても、まったく問題ないよ」と言ったなら、あなたはイカれた変人と話していると思うだろう。私が人参が好きなら、なぜ一週間食べないのか？もし私が人参が嫌いなら、なぜ私はこんな発言をする？つまり、ポルノ使用者が、一週間ポルノ無しで生き延びたという話をする時、彼らは自分を説得しているのである、（そして彼はあなたをも説得しているのである）、自分は全く問題ないよ、と。だが、彼らが何も問題がないなら、発言をする必要がない。この発言を翻訳すれば「私はなんとか一週間ポルノ無しで乗り切った」となる。ポルノ使用者は皆そうであるように、彼らはこの後、残りの生涯を乗り切ることを希望している。一週間しか生き延びれず、一週間ずっと不満でいて、その後での自慰セッションがどれほど貴重なものになるかあなたは想像できますか？

これゆえに、軽度のポルノ使用者は、重度のポルノ使用者より、より効率的に中毒なる、幻想の快楽がより大きいというだけでなく、彼らはポルノをやめる動因が減る、なぜなら彼らは時間の浪費が少なく、ゆえに、健康リスクも低い。偶に、彼らは性的機能障害になることもある、だが、彼らはその原因について確信を持たず、他の要素のせいになる。思い出してほしい、既に説明したように、ポルノ使用者が得る快楽は、探求のドーパミンサイクルと禁断症状の解消のみである。この快楽は、幻覚である、小さなポルノの怪物を、ほとんど感知できない痒みとして想像してみよう、それはほとんどの間、気付かないままだ。

ずっと消えない痒みがあるとしたら、それを掻くのは自然なことだ。報酬回路が、次第に、ドーパミンとオピオイドに対する抵抗性を増すなら、自然なこととは、じわじわ引き伸ばす、エスカレートさせる、度を越してやる、新奇さ探求、ショック探求、などである。ポルノ使用者がぶっ続けでポルノを見ることを妨げる四つの主な要因とは、

＜時間＞ほとんどの人は手にできない

＜健康＞痒みを解消するために、まずは、我々は手に入るすべての無料のものを消費する必要がある、その他にもいくつか。この種の、度を超えることへ対処能力は個人差があり、時と場合でも異なる。これは自動的な抑制として機能する。

＜規律＞社会、ポルノ使用者の仕事、友人、親戚によって課される；恐らくは、ポルノ使用者の頭の中で行われている本能的綱引きの結果として、ポルノ使用者自身が自分に課すこともある。

＜想像力＞想像力の欠如は、その人の主観に基づき動画のショックも新奇さもその他の価値も弱める。

「軽度ではない」ポルノ使用者は、弱者であり、他人がなぜその「摂取」を制限できるのか理解できない、と考えるのは簡単だ。しかしながら、重度のポルノ使用者は留意しておくべきだ、ほとんどの軽度のポルノ使用者は、単に、強い想像力とスタミナが求められるので、ぶっ続けで見ることができないだけだということを。重度のポルノ使用者が羨む傾向にある週に一回だけのポルノ使用者は、物理的にそれ以上できない、もしくは仕事や社会の制限、もしくは自分が中毒なっていることの自己嫌悪によってできない。

少し定義を提供すると良いのかもしれない。

The Non-user、ノンユーザー、 非ポルノ使用者

この罠の獲物となったことのない人で、調子に乗るべきでない人。彼らは、ただ、幸運、もしくは、善行の徳性によって、非ポルノ使用者である。すべてのポルノ使用者は絶対にポルノ中毒にはならないだろうと確信していた、そして、非ポルノ使用者の中には、偶のセッションを試そうとし続けている者もいる。

The Casual User、カジュアル・ユーザー、 軽度ポルノ使用者

これについては二つの基本的な分類がある

　１、ポルノ使用者で、罠に落ちているがそのことに気づいていない人（この様なポルノ使用者をうらやまないでほしい）。彼らはただ、食虫植物の蜜を試飲しただけ、十中八九、すぐに重度になる。思い出してほしい、アルコール依存症患者は皆、最初はカジュアルな飲酒者である、同様に、ポルノ使用者も最初はカジュアルに始める。

　２、元重度ポルノ使用者。彼らはポルノをやめることができないと考えている。このポルノ使用者は、一番悲しい人たちで、様々な分類に該当するが、それぞれに注釈が必要である。

The Once a Day User 、ワンス・ア・デイ・ユーザー、一日一回ポルノ使用者

もし彼らが、自分のオーガズムに達する特権を気に入っているなら、なぜインターネットポルノを一日に一回しか使わないのか？もし、やってもやらなくてもいいなら、何故わざわざする？思い出してほしい、この「習慣」は、実際には、自分で自分の頭を壁に叩きつけ、それをやめた時にリラックスする、というものである。一日一回ポルノ使用者は一時間以内に禁断症状の解消を済ませる。無自覚ではあるが、その日の残りの時間を彼らは壁に頭を叩きつけながら過ごす、人生のほとんどをこれをして過ごす。彼らが一日一回にするのは、捕まるリスクを犯せない、もしくは、神経学的な健康のリスクを犯せないからである。重度ポルノ使用者を、あなた達は好きでそれをやっていないと説得するのは簡単だが、軽度ポルノ使用者を説得するのはそれよりも難しくなる。削減をやったことのある人は、それが最悪のものだと知っている、一生ポルノ使用者で居続けることを、ほとんど保証するものだと。

The Rejected User、リジェクテット・ユーザー、断られたポルノ使用者

彼らは、毎日オーガズムに達する権利を求める、だが、彼らのパートナーはいつも喜んでその要求に答えるわけではない。最初、彼らはこの空白をインターネットポルノで埋めようとする、とても興奮する「ウォータースライド」に乗ってしまうと、彼らは、新奇さ、ショック、超自然的な画像などのサイクルに囚われてしまう。実際には、彼らはパートナーの断りを喜んでいる、言い訳になるから。もし、インターネットポルノがあなたにそれほどのものを与えると言うのなら、なぜわざわざパートナーを持つ？彼らを自由にしてあげればよい。彼らは、パートナーを頭の中で「キャリー（思い浮かべる？）」てしまう時、自慰セッションを楽しんですらいない。すぐに、彼らは、現実世界のパートナーに、インターネットの暗い谷を冒険するための言い訳を与えてもらうことを楽しみにするようになる。

The Porn-Diet User ポルノダイエット　削減をするポルノ使用者

「やめようと思えばいつでもやめられる。何度もそうたってきたし！」、としても知られる人たちのこと。

もし彼らがポルノダイエットをすることでパートナーを探す気になるなら、なぜ４日に一度のポルノダイエットなんだ？誰も未来を予知できない、あなたが予定している自慰セッションの一時間後に思いがけない出会いがあったらどうするんだ？また、もし、時々の、「配水管の掃除」が緊張をほぐすのに良いと言うなら、何故毎日、通水をしないのか？性器を健康に保つために自慰は必要ではないし、インターネットポルノは全く必要ではない、ということは証明されている。例えそうだとしても、ナンパ師の教祖であっても、神経学的な損傷に関する記事を読めば、スーパー刺激物であるポルノを見ること勧めないだろう。真実とは、彼らはまだ中毒だということ。彼らは身体的な依存症は除去できているが、洗脳の主な問題は残されたままである。彼らは毎回、永遠にやめることを望んでいる、だがすぐに再び同じ罠にかかる。

ほとんどのポルノ使用者は、実際に、やめた人たちや初心者たちを羨んでいる、ポルノダイエットをする人が、自分の欲求をコントロールできるなんて、なんて「幸運」なんだと考えている。しかしながら、彼らは見落としている、ポルノダイエットをする人たちは自分の欲望をコントロールしているのではない、彼らはポルノを使っている時、使いたくなかったと願っている。彼らはやめることの困難を経験し、その後で困窮を感じ始め、再び罠に嵌まり、しなければよかったのにと願う。彼らは両方の世界の最悪を得る。考えてみれば、これはポルノ使用者の人生においてもそうである、自慰セッションが認められた時、それを特権と捉えるか、でなければ、しなければよかったと願うか。ポルノが貴重になるのは、困窮を感じた時だけである。この「禁断の果実」症候群は、ポルノ使用者の酷いジレンマの一つである。彼らに勝ち目はない、なぜなら、彼らは迷信と幻想を求めて嘆いているから。唯一の勝ち筋は、ポルノをやめることによって、嘆くのをやめること！

The "I Only Watch Static/Tame/Home-Made Porn" User　「私は、静的な/管理された、単調な/ホームメイドのポルノしか見ない」ポルノ使用者

そう、みんな最初はこう言う、だが、これらの動画のショックバリュー（衝撃性）の平均値が急速に上昇するように見えることそして我々が気づく前に困窮を感じていること（耐性）、その様はすごいと思わないか？静止画のポルノには新奇さが欠けている、なので我々はpay the piper（＊報いを受ける、という慣用句だが、この場合は？）笛吹にお金を払ってひとカップの潤滑剤を買う、そして、ウォータースライドを滑り降りる、恨みと罪悪感に向かって。最悪のやらかしは、自分のパートナーの写真を（もちろん許可はとった上で）自慰に使うことである。なぜか？それは、その過程において、あなたは自分の脳を、探求と多様さに誘因されるドーパミン奔流に対し再配線をする。化学的には、脳で起こるポルノウォータースライドはデルタフォスBの増強である、なのであなたは、現実でパートナーと居る時に自分が困難を抱えていることに気付くだろう。

この分類のもう一つの罠としては、「素人」ポルノ、「ホームメイド」ポルノである。ほとんどは偽物で、あなたはそれを知っている、加えて、あなたは目に入った一番最初の動画で止まることはない、求め探し続ける。覚えておいてほしい、脳はオーガズムだけではなく、新奇さの狩りも求める、これがウォータースライダーにスリルを加える。ポルノの内容は問題点ではない、素人であろうがプロであろうが、問題点は脳の中で起きるドーパミンの奔流である、これが耐性を付けることと飽和の原因となる。ポルノは脳の正常な機能を破壊する、自慰は筋肉-脳反応を混乱させる、オーガズムは脳にオピオイドを激しく流し、次回に通り易いようにする。

The "I've Stopped But Have an Occasional Peek" User、「やめたが偶に覗き見する」ポルノ使用者

ある意味では、覗き見ポルノ使用者は最も哀れである。彼らは人生を自分は必要なものが奪われたと信じながらやり過ごすか、もしくは、より多い事例として、偶の覗き見がニ回になる。滑る坂を下りながら、遅かれ早かれ重度ポルノ使用者に戻る。彼らは、自分が最初に嵌った罠に、再び落ちる。

軽度ポルノ使用者には、さらに、ニつの分類がある。ひとつは、最新のセレブリティ（有名人）のセックステープでニュースになった画像や動画か、もしくは学校や職場での「偶然」の目撃を家に「持ち帰った」もので自慰をするタイプ。この人たちは本当にただの非ポルノ使用者であるが、機会を逃していると感じている。彼らはその行為に加わりたい、我々のほとんどが最初はこれと同じようにして始めた。しばらくしてからの次の時、自分の妄想のセレブリティがもうそれをしてくれないことに気付いてほしい（＊？、同じ興奮を与えてくれない？）。妄想の標的がより「獲得不可能」であればあるほど、オーガズムの禁断症状は苛立たしいものになる。

二つ目の分類は、最近になって注目されているネット上でシェアされている事例が一番よく描写できている。

インターネット上のポルノ物語を何年も読んでいる働いている女性で、一晩に一回以上は使ったことがない。偶然、彼女はとても意志の強い女性だった。ほとんどのポルノ使用者は、そもそも、なぜ彼女がやめようと思ったのか不思議に思い、PIED（ポルノによる勃起障害）（彼女の場合はPE（早漏））のリスクは存在しないと喜んで指摘するだろう、（真実ではなく、リスクはある）。彼女は、静止画すら使っていなかった、彼女が読んでいた物語は、その指摘をしたポルノ使用者が日々使っているものに比べれば、遥かに単調で管理されたものだった。

彼らは間違った思い込みをしている、軽度ポルノ使用者がより幸せでより自制できていると。より自制できてはいるかもしれないが、間違いなく幸せではない。この女性の場合、彼女はパートナーにも現実のセックスにも満足できず、日々のストレスと負担にとても苛立っていた。彼女の最も親密な人は、何が彼女を悩ませているのか特定できなかった。例え、彼女が理性を用いてポルノの使用を恐れることはないと自分を説得したとしても、彼女は現実の恋愛（良い時も悪い時もあり、それは避けられない）を好きになれなかっただろう。彼女の脳の報酬中枢は、生活の中にある普通のストレス解消を、毎日のドーパミン洪水の結果、利用することができなかった。その結果である脳の受容体のダウンレギュレーション（減少）は、彼女を、ほとんどの状況下で、憂鬱にさせた。ほとんどの人たちがそうであるように、彼女は、初めてのポルノを見る前、ポルノの闇の側面と女性の扱いに強い恐れを持っていた。やがて、彼女は社会的洗脳の被害者となり、最初のサイトを試す。暴力的な不快な動画を見た時、屈服し常用ポルノ使用者となるほとんどの人たちとは違い、彼女はその滑落に抗った。

あなたがポルノで楽しむのは、その前に始まった欲求（ほぼ感知できない物理的な欲求、もしくは、痒みを掻くことを許されない精神的拷問）を終わらせることだけ。インターネットポルノそのものは毒であり、ゆえにあなたは、禁欲期間の後で、それを楽しんでいるという幻覚を見るだけなのだ。空腹や喉の渇きと同じで、長く苦しむほど、解消された時の喜びは大きい。ポルノはただの習慣だと信じる間違いを犯すことで彼らは思う：「一定レベル以下、または特別な時だけの使用を維持できれば、自分の身体と脳はそれを受け入れるだろう。そうなれば、そのレベルを維持できるし、望むならさらに減らすこともできる」

思考をクリアにしてほしい、「習慣」は存在しない。ポルノは麻薬中毒である、自然な傾向として、一時的に、禁断症状を解消する。あなたの現状のレベルを維持するには、あなたは、残りの人生において、とんでもない量の規律と意志の力を求められる；あなたの脳の報酬中枢はドーパミンとオピオイドに耐性を持つので、その要求は増え続け、減ることはない。

ポルノが次第にあなたの神経系、勇気、自信、衝動の制御を破壊するので、あなたは、ますます、自慰セッションの中断時間を短くすることに抵抗できなくなる。だから初期には我々は、やってもいいしやらなくてもいい、と言っていられる。もし、我々が、身体的、もしくは、精神的に何かが間違っているという兆候を得られたなら、我々はやめる。この女性を羨まないでほしい、24時間に一度だけ見る場合、それは地上で最も貴重なものに見える、それはポルノを「禁断の果実」に変えてしまう。何年も、この可哀想な女性は、綱引きの中心にいた。

ポルノの使用をやめられず、だが、動画にエスカレートすることを恐れ。23時間と、その間の日々のすべての10分間、彼女は誘惑と自分の彼氏に対する思慮の欠如と戦わなければならなかった。彼女のしたことは、とんでもない意志の力を必要とする、最終的には彼女を弱らせ涙を流させた。この様な事例は珍しいが、論理的に見てみよう：ポルノには、本当の支えや喜びがあるのか、ないのか。もしあるならば、なぜ、一時間、あるいは一日、あるいは一週間も待つ人がいるのか？なぜ、その間、あなたの支えや喜びは奪われるのか？本当の支えや喜びがないとするならば、なぜ、わざわざ、あなたは自分のオンラインハーレムを訪れるのか？
